自带的man手册只能查询C库的函数和linux系统调用，不能查询linux内核函数。如果做驱动开发，就很不方便查询linux的内核函数。我们可以通过linux内核生成Man手册。

  • 下载linux内核源码，并且编译通过
  • 在顶层目录下执行,这一步时间有点长
make mandocs
  • 安装man手册，就是把生成的文件拷贝到系统目录下，因此需要sudo权限
sudo make installmandocs
  • 如果第一步出现错误，则执行下面的命令
sudo apt-get install xmlto
